# 🚀 Multi Utility Toolkit (Python Project)

A Python-based **Modular & Packager Project** that demonstrates the use of built-in modules, custom modules, and packages with a menu-driven interface.

---

## 📌 Features

- 📅 Date & Time Operations (datetime, time)
- 🧮 Mathematical Calculations (math)
- 🎲 Random Data Generation (random)
- 🆔 UUID Generator (uuid)
- 📂 File Handling (Custom Module)
- 🔍 Dynamic Module Exploration using dir()
